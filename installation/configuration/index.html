    위 env 파일을 복사하여 동일한 폴더에 .env 파일을 만들고 환경 설정 파일을 작성합니다.

    세션 설정, 쿠키 설정, CSRF 방지 토큰, 데이터베이스 설정 등과 관련된 항목은 CodeIgniter 공식 가이드북을 참조하십시오.

    이석찬에서 새롭게 정의되는 환경설정 항목에 대한 설명은 아래와 같습니다.

    app.baseURL
    서비스의 기본 주소입니다. http:// 또는 https://를 포함하여 작성합니다.
    예시
    app.baseURL = https://iseokchan.com
    app.baseHostname
    서비스의 기본 호스트 이름입니다. http:// 또는 https://를 제외한 부분을 작성합니다.
    예시
    app.baseHostname = iseokchan.com
    app.apiURL
    API 서버의 기본 주소입니다. http:// 또는 https://를 포함하여 작성합니다.
    예시
    app.apiURL = https://api.iseokchan.com
    app.apiHostname
    API 서버의 기본 호스트 이름입니다. http:// 또는 https://를 제외한 부분을 작성합니다.
    예시
    app.apiHostname = api.iseokchan.com
    jwt.secret
    JWT의 암호화 키입니다.
    예시
    jwt.secret = secret-key
    jwt.token_name
    JWT을 저장할 쿠키 이름입니다. app.cookiePrefix 값에 영향을 받습니다.
    예시
    jwt.token_name = access_token
    jwt.algorithm
    JWT의 암호화 방식입니다. php-jwt에서 지원하는 방식만 지정가능합니다.
    예시
    jwt.algorithm = HS256
    app.CFEnabled
    CloudFlare 서비스를 적용중이면 true, 그렇지 않으면 false를 지정합니다. true로 설정된 경우 사용자의 아이피를 X-Forwarded-For 헤더 또는 CF-Connecting-IP 헤더에서 받아옵니다.
    예시
    app.CFEnabled = true
